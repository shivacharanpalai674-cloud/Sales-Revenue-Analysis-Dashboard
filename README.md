\# Sales \& Revenue Analysis Dashboard



\## Overview

This project is an interactive Sales \& Revenue Analysis Dashboard developed using Power BI.



\## Features

\- Total Sales KPI

\- Total Quantity KPI

\- Total Profit KPI

\- Monthly Sales Trend

\- Sales by Category

\- Sales by Region

\- Top Performing Products

\- Interactive Slicers



\## Tools Used

\- Power BI

\- Microsoft Excel
